# dns-allowlists

Централизованные DNS allowlist’ы для локальной инфраструктуры (AdGuard Home).

## Назначение

Репозиторий содержит списки доменов, которые должны быть **разрешены** при использовании DNS-фильтрации и чёрных списков.

Формат списков: **один домен на строку** (FQDN), без AdBlock-синтаксиса.

Примеры корректных строк:
- `example.com`
- `api.example.com`

## Файлы

- `global.txt` — глобальный allowlist (домен разрешён для всех клиентов).
- `jellyfin.txt` — allowlist для клиента/сервиса Jellyfin (IPTV/CDN/внешние API).
- `ps5.txt` — allowlist для PlayStation 5 (авторизация/сервисы/загрузки).
- `smb.txt` — allowlist для SMB/torrent-клиента (трекеры и инфраструктура).
