# dns-policy

Персональный репозиторий DNS-политик для AdGuard Home.

## репозитории
Это личный набор DNS-списков для управления фильтрацией в домашней сети. Списки поддерживаются в актуальном состоянии под конкретную инфраструктуру.

> Репозиторий не является публичным продуктом: структура, имена файлов и состав доменов могут меняться без обратной совместимости.

## Назначение
- Централизованное хранение разрешённых и заблокированных доменов
- Версионирование изменений DNS-политик
- Быстрое развёртывание правил на нескольких экземплярах AdGuard Home
- Разделение политик по сервисам

## Формат
- Один FQDN на строку
- Без AdBlock-синтаксиса (`||`, `^`, `*`, `$`)
- Без wildcards (`*.example.com` — не поддерживается AdGuard в raw-списках)
- Комментарии только в README, не в списках
